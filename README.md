# _Mr. Roboger_

#### By _Kari Vigna_

#### _This application takes a number then transforms it into a secret message._

## Technologies Used

* _HTML_
* _CSS_
* _Javascript_
* _brain juice_

## Description

_{This application takes an integer and returns a sequence of elements from 0 to inputted integer. All elements containing a 1 will be replaced with "Beep!", elements containing a 2 will be replaced with "Boop!" and elements containing a 3 will be replaced with "Won't you be my neighbor?"}_

## Setup/Installation Requirements

* _Clone this repository to desktop._
* _Navigate this projects top level directory._
* _Open roboger/index.html in your browser._

## Known Bugs

* _Does not return "0" with sequence of numbers._
* _Only replaces first instance of 1._
